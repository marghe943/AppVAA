# Frontend Voting Advice Application

## Comandi utili

Da eseguire tutti dalla cartella root:

### Setup progetto
```
npm install
```

### Avviare server di sviluppo
```
npm run serve
```

### Compilazione e minify per la produzione
```
npm run build
```
